# Apresento o projecto subistituto
# Análise de dados de Angola
Nome do projecto: Análise dos dados do recenseamento geral da população e habitação, 2014 (questionário da habitação colectiva)
O processo incluiu:

Organização: Mapeamento de códigos para rótulos de dados e renomeação das colunas para facilitar a análise.
Limpeza: Tratamento de valores ausentes e remoção de dados inconsistentes.
Transformação: Agrupamento de categorias e conversão de variáveis para otimizar a análise.
Visualização: Importação dos dados limpos para o Power BI e criação de dashboards intuitivos para tomada de decisão.
